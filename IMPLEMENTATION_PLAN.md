## 🛠️ 技术栈对比

| 功能 | 原仓库 (Java) | Python 版本 |
|------|--------------|------------|
| 后端框架 | Spring Boot 3.5.14 | FastAPI 0.115+ |
| 数据库 | PostgreSQL | PostgreSQL/SQLite |
| ORM | MyBatis-Flex | SQLAlchemy 2.0 |
| 迁移 | Flyway | Alembic |
| AI 集成 | Spring AI | LangChain / OpenAI SDK |
| PDF 导出 | Playwright (Java) | Playwright (Python) |
| 认证 | Spring Security | JWT + FastAPI Security |
| 前端 | React 19 + Vite | React 19 + Vite (保持一致) |
