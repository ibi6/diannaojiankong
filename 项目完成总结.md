## 🌐 API 使用

### 获取系统信息
```bash
curl http://127.0.0.1:5000/api/system
```

### 获取历史数据
```bash
curl http://127.0.0.1:5000/api/history
```

### Python 调用示例
```python
import requests

response = requests.get('http://127.0.0.1:5000/api/system')
data = response.json()
print(f"CPU: {data['cpu']['percent']}%")
print(f"内存: {data['memory']['percent']}%")
```

## 💡 使用建议

### 日常监控
使用 Web 版本，在浏览器中打开 http://127.0.0.1:5000，添加到书签方便访问。

### 服务器监控
使用命令行版本，通过 SSH 远程连接后运行：
```bash
python system_monitor.py -i 5
```

### 性能测试
在运行测试程序的同时，开启监控观察资源使用情况。

### 故障排查
系统变慢时，快速启动监控查看哪个进程占用资源。

### 远程访问
启动时添加 `--host 0.0.0.0`，在手机或其他设备上访问。
