# 电脑状态监控工具

一个功能完整的系统监控工具，提供**命令行版本**和**Web 界面版本**，可以实时监控电脑的 CPU、内存、磁盘、网络等状态。

## 功能特性

- ✅ **CPU 监控**：显示总体使用率、各核心使用率、频率
- ✅ **内存监控**：显示内存使用情况、虚拟内存状态
- ✅ **磁盘监控**：显示各分区容量、使用率、读写速度
- ✅ **网络监控**：显示上传/下载速度、总流量、活动连接数
- ✅ **进程监控**：显示 CPU 占用最高的前 10 个进程
- ✅ **可视化进度条**：直观显示各项资源使用率
- ✅ **实时图表**：Web 版本提供实时历史数据图表
- ✅ **快照功能**：保存系统状态到 JSON 文件
- ✅ **自定义刷新间隔**：可调整监控刷新频率

## 两种使用模式

### 🖥️ 模式一：命令行版本（终端显示）
- 文件：`system_monitor.py`
- 适合：喜欢在终端工作的用户、服务器监控

### 🌐 模式二：Web 界面版本（浏览器显示）
- 文件：`web_monitor.py`
- 适合：需要图形化界面、多设备访问
- 特色：实时图表、现代化界面、支持远程访问

## 安装依赖

### 命令行版本
```bash
pip install psutil
```

### Web 版本（包含所有依赖）
```bash
pip install -r requirements.txt
```

或直接安装：
```bash
pip install psutil flask flask-cors
```

## 使用方法

### 🖥️ 命令行版本

#### 1. 启动实时监控（默认模式）

```bash
python system_monitor.py
```

这会启动实时监控，每 2 秒刷新一次。按 `Ctrl+C` 退出时会自动保存快照。

#### 2. 自定义刷新间隔

```bash
# 每 5 秒刷新一次
python system_monitor.py -i 5

# 每 1 秒刷新一次
python system_monitor.py --interval 1
```

#### 3. 只保存快照（不进入监控模式）

```bash
# 保存到默认文件 system_snapshot.json
python system_monitor.py -s

# 保存到指定文件
python system_monitor.py -s -o my_snapshot.json
```

#### 4. 查看帮助

```bash
python system_monitor.py -h
```

### 🌐 Web 界面版本

#### 1. 启动 Web 服务器（推荐）

```bash
python web_monitor.py
```

然后在浏览器中访问：**http://127.0.0.1:5000**

或者直接双击：`启动Web监控.bat`（Windows）

#### 2. 自定义端口和地址

```bash
# 自定义端口
python web_monitor.py -p 8080

# 允许局域网访问（其他设备可访问）
python web_monitor.py --host 0.0.0.0 -p 5000
```

#### 3. 局域网访问

如果要在手机或其他设备上查看监控：

1. 启动服务器时使用 `--host 0.0.0.0`
2. 查找本机 IP 地址（如 192.168.1.100）
3. 在其他设备浏览器访问：`http://192.168.1.100:5000`

### 快速启动（Windows）

双击以下 bat 文件即可：
- `启动监控.bat` - 命令行版本
- `启动Web监控.bat` - Web 界面版本
- `保存快照.bat` - 快速保存系统快照

## Web 界面功能

Web 版本提供了更丰富的可视化功能：

- 🎨 **现代化界面**：渐变色设计，卡片式布局
- 📊 **实时图表**：使用 Chart.js 显示 CPU 和内存历史曲线
- 📱 **响应式设计**：支持桌面、平板、手机访问
- 🔄 **自动刷新**：每 2 秒自动更新数据，无需手动刷新
- 🌐 **多设备访问**：可在局域网内任意设备访问
- 💾 **历史数据**：保存最近 60 个数据点的历史记录

### API 端点

Web 版本提供 RESTful API，可以被其他程序调用：

- `GET /api/system` - 获取当前系统状态
- `GET /api/history` - 获取历史数据（最近 60 个数据点）

示例：
```bash
curl http://127.0.0.1:5000/api/system
```

## 命令行参数

### system_monitor.py（命令行版本）

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--interval` | `-i` | 刷新间隔（秒） | 2 |
| `--snapshot` | `-s` | 只保存快照，不进入监控模式 | False |
| `--output` | `-o` | 快照输出文件名 | system_snapshot.json |

### web_monitor.py（Web 版本）

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--host` | - | 服务器地址 | 127.0.0.1 |
| `--port` | `-p` | 服务器端口 | 5000 |

## 系统要求

- Python 3.6+
- Windows、Linux 或 macOS
- psutil 库

## 注意事项

1. 首次运行需要安装 `psutil` 库
2. 某些功能可能需要管理员权限（如获取所有进程信息）
3. 在 Windows 上运行时，终端需要支持 UTF-8 编码以正确显示中文和特殊字符
4. 刷新间隔不建议设置过小（< 1 秒），可能会增加系统负担

## 故障排除

### 中文显示乱码

Windows 上如果中文显示乱码，在运行前执行：

```bash
chcp 65001
```

### 权限不足

如果某些数据无法获取，尝试以管理员身份运行：

```bash
# Windows (以管理员身份打开 PowerShell 或 CMD)
python system_monitor.py

# Linux/macOS
sudo python3 system_monitor.py
```
