## 🎯 核心功能模块

### system_monitor.py 模块
```python
class SystemMonitor:
    - __init__()              # 初始化
    - get_cpu_info()          # CPU 信息
    - get_memory_info()       # 内存信息
    - get_disk_info()         # 磁盘信息
    - get_network_info()      # 网络信息
    - get_process_info()      # 进程信息
    - display_system_info()   # 显示界面
    - save_snapshot()         # 保存快照
```

## 🔧 API 端点

### GET /api/system
返回当前系统状态（JSON）

**响应示例：**
```json
{
  "timestamp": "2026-06-05T18:19:12",
  "cpu": {
    "percent": 52.4,
    "count": 20,
    "frequency": 2300.0,
    "per_core": [0.0, 62.5, ...]
  },
  "memory": {...},
  "disk": {...},
  "network": {...},
  "processes": [...]
}
```

### GET /api/history
返回历史数据（最近 60 个数据点）

**响应示例：**
```json
{
  "cpu": [7.4, 12.3, 15.6, ...],
  "memory": [83.6, 84.1, 83.9, ...],
  "network": [...],
  "disk_io": [...],
  "timestamps": [...]
}
```
